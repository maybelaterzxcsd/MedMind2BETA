# MedMind 2.0
Система анализа медицинских текстов на базе трансформерных моделей.

## Описание
MedMind 2.0 - это инструмент для автоматического извлечения и структурирования информации из медицинских текстов на русском языке. Система использует предобученную языковую модель RuBERT для распознавания именованных сущностей (NER) и анализа клинических данных.

## Возможности
- Извлечение данных пациента (ФИО, дата рождения)
- Распознавание жалоб и симптомов
- Анализ принимаемых препаратов
- Обработка лабораторных показателей
- Сравнительный анализ между визитами (дельта-анализ)
- Проверка лекарственных взаимодействий
- Генерация структурированных отчетов

## Установка

### Требования
- Python 3.8+
- Node.js 16+

### Шаги установки
1. Клонируйте репозиторий:
```bash
git clone https://github.com/maybelaterzxcsd/MedMind2.0.git
cd MedMind2.0
```
2. Настройка бэкенда
```bash
cd backend

# Создание виртуального окружения
python -m venv venv

# Активация окружения
# Для Windows:
venv\Scripts\activate
# Для macOS/Linux:
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt
```
3. Настройка фронтенда
```bash
cd frontend

# Установка зависимостей
npm install
```
4. Загрузите модель:
Модель размещена на Hugging Face. См. раздел "Модели" ниже.

## Быстрый старт
1. Запустите бэкенд:
```bash
cd backend
python main.py
```
2. Запустите фронтенд (в новом терминале):
```bash
cd frontend
npm run dev
```
3. Откройте браузер: http://localhost:5173

## Модели
Модель `rubert_medical_v2` (2.2 GB) не включена в репозиторий из-за размера и размещена на Hugging Face:
Репозиторий модели: https://huggingface.co/maybelaterzxcsd/medmind-rubert-v2

### Загрузка модели (вручную)

1. Перейди по ссылке:  
   [https://huggingface.co/maybelaterzxcsd/medmind-rubert-v2](https://huggingface.co/maybelaterzxcsd/medmind-rubert-v2)
2. Нажми кнопку **Download** (справа вверху)
3. Скачай архив `rubert_medical_v2.zip` (или файлы по отдельности)
4. Распакуй содержимое в папку:
   MedMind2.0/models/rubert_medical_v2/
5. Структура после распаковки должна быть такой:
```bash
models/rubert_medical_v2/
├── config.json
├── model.safetensors
├── tokenizer.json
├── tokenizer_config.json
└── training_args.bin
```

## API
Бэкенд предоставляет REST API на порту 8000.
Основные эндпоинты:
- POST /api/analyze — анализ медицинского текста
- GET /api/patient/{id} — получение данных пациента
- POST /api/delta — сравнительный анализ визитов
